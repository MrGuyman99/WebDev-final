How this all Works

An abriged tutorial on how to read .CSV files

What is a .CSV file?

A .CSV file is the file format Microsoft Excel Sheets are stored in. The specific sheet we are using is formatted as the list shows. .CSV files are not very readable straight out the gate. If we look at the raw .CSV file text we get something like:

"2021","101826", "Drexel Building","Not Submitted","8136 8142 S DREXEL AVE", "60619","0.0","false", "CHATHAM",,"66,888",,,,,,,,,,,,,,,, "41.74401302","-87.60549928", "(41.74401302, -87.60549928)","2021-101826"

This isn't great. There are so many random commas, coordinates I'm sure you or I have no use for, and things like "Not Submitted" and "CHATHAM" that make absolutely no sense without context. Our goal is to take this data and condense it into something more digestable and prefereably searchable.

Property "Drexel Building" at address "8136 8142 S DREXEL AVE" in Zip-Code "60619" has an energy rating of 0.0

That's a much more digestable output, now the only question is how the hell do we get there.

Gettings There:

So, we need to take our unusable raw data and convert it into something organized and easy to access. Now, the easiest way I could think of storing this data is by utilizing an array. Now, there are a bunch of different ways to do this. I could store every Zip-Code in one array, all the Address in another, all the property names in one more, (e.t.c) and have the index of the values correspond to what row they're in. An example of this is shown at the left. Now, that's all fine and dandy, but what if we wanted to get even more organized? Well, we could store an array of arrays! Specifically, each index would store a row, and in that row we could use another index to access all the Zip-Codes, addresses, energy ratings, and property names for that row. 

// Example of using multiple arrays 
//(Assuming the Drexel Building is on row 0)
let addresses = ["8136 8142 S DREXEL AVE", ...];
let property_name = ["Drexel Building", ...];
let Zip-Code = ["60619", ...];
let energy_rating = ["0.0", ...];
// ... 
// Example of using one big array 
// CSV -> "a,b\n1,2" to
// Javascript 2D Array -> [["a", "b"], ["1", "2"]]
// Also removes extra characters like 
// "\"Lincoln Park-CPS\"" -> to Lincoln Park-CPS 
function parseCSV(content) {
    // Removes whitespaces and divides up the .CSV file by rows
    const rows = content.trim().split("\n");
    // We return a new 2D array using the .map method where we, again, make another array
    // This time going by the comma seperators.
    // Think: "Property Name", "Address" -> new Array("Property Name", "Address");
    return rows.map(row =>
        row.split(",").map(cell =>
            cell.trim().replace(/^"|"$/g, "")
        )
    );
}

let data = parseCSV("[insert file from filepicker]");
/* 
    Assuming Drexel building is in row 0, we access the 
    property-name value by using a key of 2 since the property 
    is the third element stored in row, since we read left to right
*/ 
console.log(data[0][2]); // Outputs "Drexel Building"
